    Tugas 1¶

    pencarian count,mean,std,min,max,q1,q2,q3 dan skew

    count adalah pencarian hasil akhir dari jumlah keseluruhan data

    code program

    import pandas as dt
df=dt.read_csv("contoh.csv")
df['TINGGI'].describe()
df['BB'].describe()
df['UMUR'].describe()
df['NILAI'].describe()
print("jumlah Tinggi  :",df['TINGGI'].count())
print("jumlah B.Badan :",df['BB'].count())
print("jumlah  UMUR   :",df['UMUR'].count())
print("jumlah NILAI   :",df['NILAI'].count())

    output

    varian Tinggi  : 499
varian B.Badan : 499
varian  UMUR   : 499
varian NILAI   : 499

    mean adalah nilai rata-rata yang dihasilkan dari jumlah nilai data di bagi jumlah banyaknya data.

    rumus

    code program

    import pandas as dt
df=dt.read_csv("contoh.csv")
df['TINGGI'].describe()
df['BB'].describe()
df['UMUR'].describe()
df['NILAI'].describe()
print("Rata-rata Tinggi",df['TINGGI'].mean())
print("Rata-rata Berat Badan",df['BB'].mean())
print("Rata-rata UMUR",df['UMUR'].mean())
print("Rata-rata NILAI",df['NILAI'].mean())

    output

    Rata-rata Tinggi 170.45090180360722
Rata-rata Berat Badan 119.54308617234469
Rata-rata UMUR 43.30861723446894
Rata-rata NILAI 74.6753507014028

    skew adalah nilai yang berada ditengah-tengah banyak data terurut dari terkecil hingga terbesar

    code program

    import pandas as dt
df=dt.read_csv("contoh.csv")
df['TINGGI'].describe()
df['BB'].describe()
df['UMUR'].describe()
df['NILAI'].describe()
print("kemiringan Tinggi  :",df['TINGGI'].skew())
print("kemiringan B.Badan :",df['BB'].skew())
print("kemiringan  UMUR   :",df['UMUR'].skew())
print("kemiringan NILAI   :",df['NILAI'].skew())

    output

    kemiringan Tinggi  : -0.016248849780930406
kemiringan B.Badan : 0.007341864521398289
kemiringan  UMUR   : -0.10397194858469867
kemiringan NILAI   : 0.03744260547804399

    modus adalah nilai yang sering muncul atau frekuensi yang seering terjadi pada banyaknya data.

    code program

    import pandas as dt
df=dt.read_csv("contoh.csv")
df['TINGGI'].describe()
df['BB'].describe()
df['UMUR'].describe()
df['NILAI'].describe()
print("modus Tinggi     ",df['TINGGI'].mode())
print("modus B.Badan    ",df['BB'].mode()) 
print("modus UMUR       ",df['UMUR'].mode())
print("modus NILAI      ",df['NILAI'].mode())

    output

    modus Tinggi          176
modus B.Badan         52
modus UMUR            53
modus NILAI           100
